# babel-preset-modern

> Babel preset for modern browsers. All you need to compile your code to a ES5/ES6 hybrid.

[![NPM Version](http://img.shields.io/npm/v/babel-preset-modern.svg?style=flat-square)](https://www.npmjs.org/package/babel-preset-modern)
![Edge 13](https://img.shields.io/badge/Edge-13-green.svg?style=flat-square)
![Firefox 45](https://img.shields.io/badge/Firefox-45-green.svg?style=flat-square)
![Chrome 49](https://img.shields.io/badge/Chrome-49-green.svg?style=flat-square)
![Opera 36](https://img.shields.io/badge/Opera-36-green.svg?style=flat-square)

## Presets

- modern
- modern/webpack2 use this instead if you use webpack
- modern/safari9 ![Safari 9](https://img.shields.io/badge/Safari-9-green.svg?style=flat-square) Add this if you need better safari9 support
- modern/stage1 add stage 1 or higher features

## Installation

```sh
$ npm install --save-dev babel-preset-modern
```

## Usage

Add the following line to your `.babelrc` file:

```json
{
  "presets": ["modern", "modern/safari9", "modern/stage1"]
}
```
